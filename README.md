Генерация картинок, чат с AI, решение уравнений.

 Быстрый старт
```bash
git clone https://github.com/ВАШ_НИК/MyBot.git
cd MyBot
pip install -r requirements.txt
 создайте .env с токеном
python bot.py
```

 Команды бота
/start — приветствие  
/generate_image кот в скафандре — картинка по тексту  
(остальное — через кнопки в чате)
